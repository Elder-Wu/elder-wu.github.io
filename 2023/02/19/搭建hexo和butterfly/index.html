搭建Hexo ➕ 配置Butterfly主题

发表于2023-02-19|更新于2023-02-22
|字数总计:300|阅读时长:1分钟|阅读量:

安装流程

  • 安装和配置都很简单，直接去官网看教程就行

更新流程

- Hexo直接通过homebrew更新

- Butterfly主题需要手动更新

  1. 进入hexo项目根目录，将theme/butterfly整个主题文件删除
  2. 重新git clone butterfly仓库，要确路径跟之前的一致
  3. 然后删除butterfly目录下与git版本控制相关的东西(.git文件夹)
  4. 升級完成後，到 Github 的 Releases 界面 或者 【官方文档-文檔七】 查看新版的更新內容。
    裏面有標注 _config 文件的變更內容（如有），請根據實際情況更新你的配置內容

需要注意的点

  1. 项目中用到了站内搜索algolia，所以每次部署网站之前，都要执行下hexo algolia这个命令，把本地的文档同步到远程
  2. 后面有空了会补充一个功能：调用api自动生成文章的description，tag和category，有利于seo优化，同时ai生成文章概要的方式也比较高效

这个主题东西太多，会加载很多用不到的资源，影响体验。搭建完成后，使用在线分析工具，测一下访问速度
